🚴 Smart Bike Backend

Smart Bike is a Spring Boot backend system designed to monitor and manage smart bicycle devices through MQTT communication and MySQL database storage.
The system receives sensor data from bikes, processes it, and stores it for monitoring and analytics.

📌 Features

📡 MQTT Integration for receiving IoT bike data

🗄 MySQL Database for persistent storage

⚙ Spring Boot REST API

🔄 Hibernate/JPA ORM for database operations

🧠 Designed for IoT Smart Bike systems

🏗 Tech Stack

Java 17+

Spring Boot

Spring Data JPA (Hibernate)

MySQL

MQTT (IoT communication)

Maven / Gradle
